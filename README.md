# 2D Time Loop Escape Platformer Game

## Overview
This project is a 2D Time Loop Escape Platformer game developed in Unity. The game features pixel art graphics and is designed for mobile platforms.

## Game Introduction
In this game, players control a character who can manipulate time to escape various challenging situations within a beautifully crafted world. The pixel art style gives the game a nostalgic feel while providing a modern gaming experience.

## Gameplay Mechanics
- **Time Manipulation:** Players can rewind time to undo mistakes and navigate through complex puzzles.
- **Platforming Elements:** Traditional platformer mechanics such as jumping, climbing, and dodging obstacles are featured.
- **Collectibles:** Players can collect items to enhance their abilities and unlock new levels.

## Art Style
The game utilizes a charming pixel art style, blending colorful visuals with retro aesthetics, which appeals to both nostalgia and modern gaming sensibilities.

## Development Setup
To get started with development:
1. Have Unity installed (preferably the latest version).
2. Clone the repository:
   ```bash
   git clone https://github.com/madevkumar/mobile-adventure.git
   ```
3. Open the project in Unity and follow the instructions in the README of the Unity project to set it up.

## Technical Details
- **Engine:** Unity 3D
- **Programming Language:** C#
- **Supported Platforms:** iOS, Android

## How to Play
- Use the on-screen controls to navigate the player through the levels.
- Collect items and avoid enemies to progress through the game.

## Future Directions
